# Address minor CLI and rendering polish items

## Summary

Small polish items from a June 2026 code review, grouped because each is a
one-to-three-line fix. Fix or explicitly decline each with a note.

## Items

1. **Misleading embed pattern.** `//go:embed workflow/**`
   (`internal/templates/templates.go:13`) — Go embed has no recursive `**`;
   the pattern behaves like `workflow/*` and only works because the directory
   is flat. Use `workflow` (recursive, idiomatic) or `workflow/*` to state
   intent.
2. **Misleading subcommand error.** `ahm task foo` reports "task requires a
   subcommand" even though one was given (`task_commands.go:26-29`). Report
   `unknown subcommand "foo"` instead. Same pattern in `task dep`
   (`task_deps.go:16`) and `agents` (`cli.go:180`).
3. **Exact numeric ID match loses to ambiguity.** `resolveTaskFromTasks`
   (`task_commands.go:719-747`): `ahm task show 1` errors as ambiguous when
   both `001` and `001a` exist, even though `001` is the exact numeric match.
   Prefer the exact numeric+suffix match before prefix matching.
4. **Bracket escaping in index tables.** `escapeCell` (`indexes.go:297-304`)
   does not escape `[` / `]`, so bracketed titles can mangle generated table
   links. Follow-up to task 024, which added backtick/angle-bracket/pipe
   escaping.

## Acceptance Notes
