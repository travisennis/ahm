# Simplify cleanupStaleTemps logic with identical branches

## Summary

`cleanupStaleTemps` in `internal/ahm/write.go` has an `if`/`else` structure where both branches call `os.Remove(cleanPath)`. The comments describe two different removal scenarios ("the real file exists, the .tmp is stale from a crash" vs "the real file doesn't exist, the .tmp is an orphan"), but the code does the same thing in both cases. The conditional structure implies a guard that doesn't exist.

## Files in scope

- `internal/ahm/write.go` — the `cleanupStaleTemps` function (lines 108–155)

## Files out of scope

- Any test file or other source file

## Current code (lines 139-153)

```go
origPath := strings.TrimSuffix(cleanPath, ".tmp")
origStat, origErr := os.Stat(origPath)
if origErr == nil && origStat.Mode().IsRegular() {
    return os.Remove(cleanPath) //nolint:gosec
}
if os.IsNotExist(origErr) {
    return os.Remove(cleanPath) //nolint:gosec
}
return nil
```

## Steps

### Step 1: Simplify to a single condition

Replace the two branches with one:

```go
origPath := strings.TrimSuffix(cleanPath, ".tmp")
_, origErr := os.Stat(origPath)
if origErr != nil && !os.IsNotExist(origErr) {
    // Stat failed for a reason other than "doesn't exist" — don't touch it.
    return nil
}
// Target file exists (stale .tmp from crash/interrupted write) or doesn't
// exist (orphan .tmp). Either is safe to remove.
return os.Remove(cleanPath) //nolint:gosec // best-effort cleanup within .agents/
```

Key changes:
- Single guard: only skip removal when `os.Stat` returns a non-NotExist error.
- Removed the unused `origStat` variable.
- Single `//nolint:gosec` comment instead of duplicate.
- Single comment explaining both cases.
